    k8s runs on bare metal so your container is like a VM on a hypervisor. A pod is a a group of container to make up an app, in our case is the hermes agent, then to extend functionality I use sidecar containers to host cron jobs that run on a schedule and use the agent to send me info to What's app. My goal is to separate the skills and scripts/code from the agent so I can do code security scans and to decouple components so that I don't keep adding things to the main agent.
