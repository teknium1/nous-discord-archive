    What We Launched
    
    A zero-trust data access layer for AI agents. Expanso pipelines sit between an agent and sensitive data, enforcing principle of least privilege at the infrastructure level. The agent calls an HTTP endpoint, the pipeline filters and transforms the data, and returns only what’s needed. No raw credentials, no unfiltered access.
    
    We demonstrated this integration with OpenClaw, but show support for anything, to demonstrate and distribute this. The launch includes:
    
    Blog post: “Your AI Agents Are Running Naked” — https://expanso.io/blog/2026-002-your-ai-agents-are-running-naked/
    
    Landing page: https://expanso.io/expanso-hearts-openclaw
    
    3 demo videos showing real-world use cases:
    - Classified data extraction (PII filtering): https://youtu.be/-oXUU52CI48
    - Log intelligence (99.9% data reduction): https://youtu.be/G03jG2ta8hU
    - Multi-source MultiAPI secure aggregation: https://youtu.be/vcS4QNDXA5A1 
    
    “Your AI agents have raw DB credentials, API keys, and full file access. That’s not a feature — it’s a liability.”
