    solo Rust project, 1000+ GitHub stars, 1.1k+ weekly npm installs, 100+ cloud users in the first 6 weeks since shipping.
    
    what it does: real web access for Claude, Cursor, Windsurf and any other MCP client. the agent can scrape any URL, crawl entire sites, run web searches, extract structured data with a JSON schema, track content changes between snapshots. 12 tools total.
    
    the main problem it solves is that most real websites block AI agents on the first request. webclaw handles TLS fingerprinting at protocol level so the request looks like a real browser, not a bot. works on the majority of Cloudflare-protected sites without extra config.
    
    free to use: the MCP server is open source (AGPL-3) and runs locally with zero API key needed. there is also a cloud API for JavaScript-heavy sites and harder anti-bot cases, free tier available.
    
    security note: the MCP server runs on your machine and makes HTTP requests to URLs your agent passes to it. it does not store URLs or page content. if you use the cloud API, requests go through webclaw's servers. no credentials are accessed or stored.
