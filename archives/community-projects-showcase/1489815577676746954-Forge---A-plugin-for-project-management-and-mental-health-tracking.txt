    Hey guys 👋
    
    I’ve been building Forge, an OpenClaw and Hermes agent plugin that gives agents a more structured workspace for ongoing work and psychologically relevant state.
    
    Forge is meant to complement the usual open-ended natural-language memory and reasoning. That kind of memory is great for nuance, rich context, and flexible expression. Forge adds a more explicit operational layer alongside it, so agents can persist, inspect, update, and act on structured units of work and mental-health-related state over time.
    
    With Forge 0.2.19, the plugin now supports:
    
    - goals, projects, tasks, and live task tracking
    - Kanban state
    - agent insights
    - a gamified progress layer with XP
    - habits
    - structured notes
    - event management
    - iCloud and Google Calendar sync
    
    It also includes a mental health module for working with structured records like values, beliefs, patterns, modes, and self-observation reports.
    
    That part is informed by 3rd wave CBT, especially ACT and Schema Therapy. The goal is practical rather than clinical. It is meant to help the agent keep track of what matters, what keeps recurring, what tends to drive avoidance or misalignment, and which actions are more coherent with values and longer-term direction.
    
    In practice, Forge lets agents manage goals, projects, tasks, habits, notes, and events as explicit objects and connect psychological patterns to concrete work, decisions, and next actions. You can work either through chat or through a dedicated Forge UI when visual inspection is more useful.
    
    The feedback I’d most value is:
    - what seems genuinely useful
    - any installation, setup, or onboarding friction
    
    I’d love to collaborate with anyone interested in the project. Thanks 🙂

    Forge 0.2.60 is live.
    
    Since 0.2.34, Forge has added a full rewards system with XP, action points, levels, equipment, reward history, and selectable visual themes. The large reward images are now optional downloads, so the plugin stays smaller and users can choose the art style they want from inside Forge.
    
    This update also adds much stronger agent support: live agent sessions, safer local access tokens, clearer task handoff, scoped read controls, bootstrap settings, and better agent guidance for goals, tasks, habits, notes, planning, memory, and Psyche work.
    
    The wiki and memory system were upgraded with SQLite-backed storage, migration support for old wiki files, safer shared memory handling, and better knowledge graph support for wiki pages.
    
    Daily use also got a lot of improvements: better task timers, task details, work logging, goal/project editing, settings, calendar flows, habit check-ins, overview cards, and Psyche pages.
    
    Health and companion sync got major fixes too, including better sleep data, raw sleep details, workout handling, HealthKit stability, movement timelines, stay labeling, overlap handling, and iOS companion releases.
