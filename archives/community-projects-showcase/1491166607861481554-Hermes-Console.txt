    A local-first dashboard for your Hermes setup
    
    Open source web UI that reads your local Hermes state directly from disk and makes it legible at a glance.
    
    **What it does:**
    Runtime overview - gateway health, connected platforms, update drift, diagnostics
    Session history - browse, filter, search across agents and platforms
    Crons - status, schedules, recent outputs, health indicators
    Skills browser - View skills, categories, linked files, detail views
    Memory - MEMORY.md / USER.md visibility with pressure warnings
    Key files - config and instruction file previews
    Usage - token counts and cost estimates from local session data
    
    **What it isn't:**
    
    Not a chat client
    Not a terminal replacement
    Not a multi-agent platform
    
    Reads from `~/.hermes` by default. No external services, no auth, no cloud dependency.
    
    Repo: https://github.com/shan8851/hermes-console
