    in my pr I added support for 3 types of interfaces : 
    * HTML (using jinja2 to extract all chat-rounds and show them to the user) (using markdown section for better rendering of codes) 
    * chat interface (basic gradio chat interface)
    * basic json preview (original data is json structured) 
    
    I also noticed that sharegpt is not in a standard format so i took the liberty of adding a checkbox that cleans the chat.
