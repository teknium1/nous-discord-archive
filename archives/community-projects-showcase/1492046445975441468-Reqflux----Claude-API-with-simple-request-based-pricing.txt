    Why I built it:
    
    • Token billing is unpredictable, especially with large context windows
    • Wanted a simpler way to use Claude in my projects
    
    What you get:
    
    • OpenAI-compatible endpoint (drop-in replacement)

    • Works with Claude Code out of the box
