    > What if your agents could share knowledge through a Discord voice channel while you sleep?
    
    coyote encodes data into audio that survives Opus compression with zero errors. Discord voice uses Opus natively — so coyote data transmits as valid audio without any special infrastructure. 7,900 bps at 128kbps, ~59KB per minute of audio.
    Ships with a Hermes Agent skill, Discord voice bridge, CLI, and Python API. Optional 46K-param neural decoder for noisy channels.
    `hermes skills install github:Kaden-Schutt/coyote/skills/yote` or `pip install yote`
