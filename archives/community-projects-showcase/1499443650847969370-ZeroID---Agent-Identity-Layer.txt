    One of the problems I run into with Hermes is high cost when context windows fill up, one method to fix this is heavy use of sub-agents, but permissions/scope delegation to the sub-agent is often problematic. ZeroID is an agent identity layer that uses RFC 8693 token exchange to handle scope delegation with sub-agents. Check it out here https://github.com/highflame-ai/zeroid. It is fully open source. I am looking to add a Hermes integration for this, I would welcome any help from this community
