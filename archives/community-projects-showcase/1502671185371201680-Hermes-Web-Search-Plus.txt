    📣 **hermes-web-search-plus v1.8.1 is out**
    
    A Hermes Agent plugin that adds multi-provider web search, clean URL extraction, and optional cited answers.
    
    The idea: give Hermes agents one solid set of web tools instead of hand-wiring every provider or betting everything on a single “magic” search API.
    
    🔧 **The tools**
    
    • `web_search_plus` — multi-provider search with routing, fallback, dedup, research mode, and optional quality diagnostics.
    
    • `web_extract_plus` — clean markdown/HTML extraction from URLs.
    
    • `web_answer_plus` — optional beta. Cited answer synthesis for when you explicitly want a written answer instead of search results.
    
    🚀 **Onboarding is much easier now**
    
    There’s a standalone setup helper, so you don’t need unreleased Hermes core bits to get going.
    
    Check provider status:
    `python ~/.hermes/plugins/web-search-plus/setup.py status`
    
    Guided setup:
    `python ~/.hermes/plugins/web-search-plus/setup.py setup`
    
    Or jump in with the recommended preset:
    `python ~/.hermes/plugins/web-search-plus/setup.py setup --preset starter`
    
    Starter preset = **Tavily + Linkup + Brave**.
    
    That’s a practical baseline for search, extraction, and cited-answer experiments without wiring up every provider on day one.
    
    🌐 **Search providers**
    
    Serper, Brave, Tavily, Exa, Querit, Linkup, Firecrawl, Perplexity, You.com, SearXNG.
    
    📄 **Extraction providers**
    
    Linkup, Firecrawl, Tavily, Exa, You.com.
    
    🧠 **A note on `web_answer_plus`**
    
    It’s intentionally optional beta. `web_search_plus` stays the recommended default for source discovery, current events, prices, weather, schedules.
    
    Reach for `web_answer_plus` when you specifically want a short cited synthesis.
