    Hermes-Agent: Native Web3 Intelligence Toolset
    
    I’ve just submitted a PR to add a native Web3 intelligence toolset to the Hermes core. This allows Hermes to interact directly with the blockchain without relying on external search engines.
    
    Key features:
    
    Contract Auditor: Checks if a smart contract is verified on Etherscan.
    
    Wallet Profiler: Fetches real-time ETH balances for any EVM address.
    
    Gas Sentinel: Tracks live network gas prices (Safe/Fast Gwei).
    
    Price Oracle: Gets live asset pricing via CoinGecko integration.
    
    You can see the video below showing Hermes autonomously using these tools to generate a local report.
