    yo. beta's ready, looking for a brave few 'fore i ship publically
    
      here's what's hermetica can do for you: 
      - **tldr**: run your hermes agent on your server full parity on your phone via bridge using a retro-inspired TUI-based interface: as many profiles as you have, as well as other REPLs/agents like Claude and Codex
      - **bilateral sync between bridge and app **— history caches on-device, updates both ways
      - **full session & agent management **— create, resume, rename, delete, search, switch agents, jump to the latest writable session per agent
      - **command parity with desktop Hermes** — model picker, categorized command inventory via a spiffy UI or via slash command, with display of skills toggleable, all in-app
      - **Codex and Claude Code integration** - sessions managed in the same list with runtime metadata, command parity
      - **Gemini and opencode integrations** coming soon*
      - **highly customizable**, including toggling the unhinged spinner text in my screenshots
      - **copy function for agent output** per output block
      - **voice input** - speak, serenade, or croon to your agent
      - **CRT display effects, fully toggleable **— subtle to fully cooked, your call
      - **integrated peon-ping **featuring some favorites like SCV, kerrigan, solid snake
      - **bridge diagnostics**, log viewer, MCP reload /restart in-app
      - **four** dope app icons
    
    setup: one curl command from the repo — handles the bridge, generates a token, configures Tailscale Serve, prints the URL + token to enter on your phone. takes two minutes if Tailscale's already running.
    iOS 17+, sideload via Sideloadly
