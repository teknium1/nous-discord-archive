    Built a tool to manage llama-server if anyone's tired of killing processes and restarting with different flags.
    
    Rookery — Rust daemon + CLI for local inference. Swap models without downtime, monitor GPU, manage agents like Hermes.
    
      rookery swap gemma4_moe    # hot-swap to Gemma 4 MoE
      rookery swap qwen_dense    # switch to Qwen3.5 dense
      rookery status             # GPU stats, model info, health
      rookery agent start hermes # launches + auto-restarts on crash
    
      - Define model profiles (quant, context, KV cache, sampling) in a config file
      - Hot-swap between profiles — graceful shutdown, health check, ready
      - Agent watchdog — auto-restart on crash, bounce on model swap, dependency port health
      - GPU monitoring via NVML
      - Web dashboard on LAN
      - OpenAI-compatible API passthrough
    
      Running it 24/7 on an RTX 5090 with Hermes + Gemma 4 MoE Q8_0. Hermes restarts automatically when I swap models.
    
      https://github.com/lance0/rookery
