    Hey all, my friend has made a tool for using the latest voice models for making multi voice audiobooks with dialogue detection. Much better than any cli tool I was using myself.
    https://github.com/kenning/BookForge-Studio
    You can use any model via ollama (hermes) for the dialogue detection, then assign voices from the different tts models to the different entities. Rattle out some audiobooks for the xmas break?
    Thought it might be of interest to y'all, as it has a nice UX for versioning if the models output weirdness (why are tts models so weird?). Plus I did a couple of commits for the proj.
