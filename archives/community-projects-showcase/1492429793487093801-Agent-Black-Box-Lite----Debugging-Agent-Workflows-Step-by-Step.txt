    I noticed that debugging agent workflows is still pretty opaque.
    
    So I built a small tool that logs and replays agent steps — including decisions, tool usage, and outputs.
    
    It acts like a lightweight “black box” for agents.
    
    Example output:
    
    Step 1:
    User Input → Analyze this repo  
    Agent Decision → Use file_search tool  
    Tool Used → file_search  
    Tool Output → Found relevant files  
    
    Step 2:
    User Input → What files are there?  
    Final Response → Repo contains storage.py, logger.py, replay.py, cli.py  
    
    It’s fully local, minimal, and designed with Hermes-style workflows in mind.
    
    Repo: https://github.com/Doppler2u/agent-blackbox-lite
    
    I also added a small example of how this could integrate into Hermes workflows.
    
    Would love feedback or ideas on how this could be improved or integrated further.
