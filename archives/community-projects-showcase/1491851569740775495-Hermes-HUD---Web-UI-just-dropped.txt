    The browser version of the TUI consciousness monitor. Same data, same soul, rendered for the web.
    
    What it does:
    Reads from `~/.hermes/` and shows everything your agent knows about itself, memory capacity, skills acquired, session history, tool usage patterns, service health, cron jobs, project activity, agent profiles, and more.
    
    What's new vs the TUI:
    - Dedicated Memory, Skills, Sessions tabs
    - Per model token cost tracking with daily trend sparklines
    - Command palette (Ctrl+K) for fuzzy navigation
    - 4 color themes with live preview and CRT scanlines
    - Responsive layout (works on phone too)
    
    Please let me know what you think about it or any issues you encounter. Feel free to send PRs and issues. Thank you. 
    
    PS: more improvements coming around fonts and readability.
