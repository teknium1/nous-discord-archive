    Exocortex is a local memory + coordination layer for AI coding agents like Claude Code, Codex, and Hermes. It lets multiple agents share memory, hand off work, avoid stepping on each other’s files, and keep an audit trail of what happened. It is local-first, with SQLite-backed memory, policy-gated tools, isolated git worktrees, an append-only audit log, a 31-tool MCP server.
