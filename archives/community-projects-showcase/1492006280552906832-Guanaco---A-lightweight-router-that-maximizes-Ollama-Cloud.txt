    Guanaco - An OpenAI-compatible LLM proxy that maximizes your Ollama Cloud subscription.
    
    Recently Ollama Cloud has had shaky api uptime, especially with newer models, this solves that with good fallback support.
    
    Features:
    - Drop-in LLM router (OpenAI + Anthropic compatible)
    - Search & scrape API endpoint emulation (Tavily, Exa, Firecrawl, Jina, Serper, Brave, SearXNG, Cohere)
    - Rerank support (Jina + Cohere compatible)
    - Automatic fallback to secondary coding plan provider
    - Real-time usage & quota tracking
    - API key management
    - Web dashboard
    - One-command install
    
    
    Works great with hermes and other harnesses.
