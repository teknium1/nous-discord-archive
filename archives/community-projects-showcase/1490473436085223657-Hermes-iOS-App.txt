    Building a Hermes iOS app + desktop companion to give my agent some extra iOS tools
    
    Some features I’ve built already: 
    
     -Basic chat interface
    
    - Integrated OpenAI’s Realtime STS Voice Model for Voice Mode (live camera, Hermes tool calling, maybe CarPlay or widget next?)
    
    - Sensor pipeline for health, location services. Can now ask agent health or location specific questions. “How many steps did I walk yesterday? “What’s a good coffee shop nearby?”
    
    - Relay for APN background push notifications
    
    Still very much a WIP, but I’ll make the repo public soon when it’s in a state for sharing. Plan to have a little fun with polishing the UI and fixing a few bugs first.
    
    Let me know if you any ideas or suggestions 🙂
