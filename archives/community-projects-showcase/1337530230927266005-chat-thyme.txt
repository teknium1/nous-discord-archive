    hello friends, I want to share something I built: https://github.com/chilir/chat-thyme.
    
    chat-thyme is a system for setting up Discord bots interfacing with any LLM framework with OpenAI compatibility. Search capabilities (w/ Exa) is also available for any models that support tool use. The Exa search demo in the README is using `hermes3:8b-llama3.1-q6_K`  served through Ollama on a M1 Macbook Pro.
    
    Fully open source (MIT), please check it out if you're interested 😄
