    In my context:
    (~78 skills, ~10K chars, ~3,200 tokens) into the system prompt on every API call. 
    Even "Hi" pays this cost. 
    With 78 skills, the skills index is 77% of the total payload. 
    This is wasteful, the agent rarely needs more than 2-3 skills per task, and most calls need zero.

    Let's call this "SKILL OVERFLOW"
