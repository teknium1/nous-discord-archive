    I made a small dashboard plugin for Hermes Agent <:nousgirl:1341983870752653352> memory:
    👉 https://github.com/xraysight/hermes-memory-ui
    
    It provides a simple UI to view and work with Hermes memory data more comfortably. I built it mainly for my own use, but maybe it will be useful for others too. It displays built-in memory and active external memory provider (currently Holographic, Mem0, Honcho and Hindsight). The plugin can be possible extended to other external memory providers.
    
    In case of built-in it nicely shows limits of USER and MEMORY files. For holographic I like to see categories and trust level.
    
    I am not a professional developer, so this work is from Hermes for Hermes 😄. If you use a different memory providers such as honcho  or hindsight feel free to contribute.
    
    EDIT: Added Mem0 support.
    EDIT2: Added Honcho support.
    EDIT3: Added Hindsight support.
