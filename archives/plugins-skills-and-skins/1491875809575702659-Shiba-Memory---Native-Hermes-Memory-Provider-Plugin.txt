    Hey everyone! Just open sourced my first project. 
    
    Shiba Memory is a persistent, self-improving memory layer for AI agents. It ships with a native Hermes memory provider plugin out of the box.
    
    What Hermes gets:
    - shiba_recall / shiba_remember / shiba_forget tools available to the LLM
    - Auto memory on every conversation turn
    - Relevant memories prefetched before each response
    - Session summaries extracted when sessions end
    - Tier 1 pattern extraction ("I prefer...", "Remember that...") + Tier 2 LLM-based summarization
    
    Fully local, no cloud deps. Postgres + pgvector + Ollama under the hood.
    
    I've been running it with Gemma 4 on a local server and it's held up well but would love for people to stress test it with Hermes and different models. Drop issues, findings, or feedback here or on GitHub.
