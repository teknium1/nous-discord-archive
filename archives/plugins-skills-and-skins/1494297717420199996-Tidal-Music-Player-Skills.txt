    Being an avid music lover and using Tidal I've created this skill set up:
    
    TIDAL SKILLS OVERVIEW
    
    1. tidal-curator (Base Skill)
    Purpose: Foundation for all Tidal integration  
    What it does:
    - OAuth authentication handling
    - Playlist creation/management
    - Track search and addition
    - Rate limiting protection
    
    Used by: All other Tidal skills
    
    ---
    
    2. tidal-dual-curator (Daily Automation)
    Purpose: Automated twice-daily music curation  
    Schedule: 9am (morning vibe) + 6pm (evening vibe)
    
    Features:
    - Curated track database with context
    - Time-of-day matching (energetic morning, chill evening)
    - Artist background and "why this track" explanations
    - No duplicates (state tracking)
    - Cron-scheduled
    
    Track database includes: Mac Miller, Kid Cudi, Aminé, East Forest, Horrorshow, Satsang, etc.
    
    ---
    
    3. tidal-playlist-creator (Bulk/Export)
    Purpose: Create playlists from track lists or ListenBrainz data  
    Three approaches:
    1. Daily curated additions (rate-limit safe, recommended)
    2. Direct Tidal API (bulk, needs 2s delays)
    3. Export to third-party (CSV/M3U/JSON for TuneMyMusic/Soundiiz)
    
    Key learning: Tidal API has strict rate limits (HTTP 412 errors if too fast)
    
    ---
    
    4. tidal-recommendation (Smart Discovery)
    Purpose: ListenBrainz-powered recommendations with web search  
    User: splosh123 (ListenBrainz)
    
    Two modes:
    1. On-demand: "Recommend me something funky" → web search + analysis
    2. Scheduled: Daily discovery cron jobs
    
    Discovery sources:
    - ListenBrainz listening history analysis
    - Web search for new releases
    - Music blog crawling
    - Similar artist exploration
    - Underground trending tracks
    
    Output: Rich context (why this track, artist background, mood classification)
    
    ---
    
    CURRENT SETUP STATUS
    
    | Skill  | Automation |
    |-------|---------|
    | tidal-curator | Base library |
    | tidal-dual-curator | Cron: 9am & 6pm daily |
    | tidal-playlist-creator | On-demand |
    | tidal-recommendation |  Cron + on-demand |
    
    Next task: Create ASCII Album art with each music recommendation!
