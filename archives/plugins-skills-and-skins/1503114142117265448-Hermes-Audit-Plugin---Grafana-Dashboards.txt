    I created a small Hermes plugin that records every tool call and every LLM API call into a local SQLite database (one DB per profile), with five ready-to-go Grafana dashboards   bundled:                                                                                                                                                                   
                                                                                                                                                                    
      - Tool Calls — success rate, errors, latency percentiles, slowest calls                               
      - API Calls — tokens, durations, finish reasons per call
      - Sessions — activity rollup per session_id                                            
      - Models — per-provider / per-model usage and token totals
      - Skills — skill_view / skill_manage activity per skill                                                                                                                    
                                                                                                                   
      Self-contained docker compose setup for Grafana included.   Not a Langfuse replacement — just a lightweight way to see what your  
      agent has actually been doing.
